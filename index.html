AP PT CodePrint

A one-stop shopping browser tool for preparing your code to submit for the AP CSP Performance Tasks. If you can print a PDF from the browser, this should be everything you need.








Steps:
  1. Set font size and draw shapes on the code.
  2. Beautify Code, it will do nice code formatting.
  3. Check your code, make sure everything is correct.
  4. Hide the controls.
  5. Print to PDF from the browser.
Note: It is highly recommended that you only use one rect and one oval for the AP Performance task to avoid confusing the reader. However, in some circumstances your abstraction or algorithm may get split up, requiring you to need more than one shape to mark the appropriate sections.
